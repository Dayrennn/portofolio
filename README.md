# Portofolio

Ini adalah project portofolio web berbasis React.js yang sedang saya kembangkan untuk menampilkan kemampuan frontend development, sekaligus sebagai dokumentasi perjalanan belajar React dan fullstack development.

Saya menggunakan Vercel sebagai tempat Hosting gratis
Link untuk akses portofolio :

```bash
portofolio-vert-one-82.vercel.app

```

Project Purpose

1. Tujuan utama pengembangan project ini adalah:
2. Membuat portofolio online menggunakan React.js
3. Menguasai component-based architecture
4. Memahami state, props, dan hooks di React
5. Menyusun project React dengan struktur yang rapi dan scalable
6. Menjadi dokumentasi kemampuan frontend / fullstack development

Bagian - bagian dari portofolio :

1. Homepage / Landing Page
   Tampilan awal dengan informasi singkat tentang saya

2. Projects / Portfolio Section
   Menampilkan project-project yang pernah dibuat (frontend & backend)

3. About Me Section
   Profil, skill, dan pengalaman

Tech Stack

1. React.js
2. JavaScript (ES6+)
3. HTML5 & CSS3
4. React Router (untuk navigasi antar halaman)
5. Vite / Create React App (sesuai setup)

Notes

2. Sudah di deploy
